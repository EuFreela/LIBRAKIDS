App Características

O aplicativo LIBRASKIDS, inicialmente, é compatível com sistemas operacionais Android para versões ^5.1.

Dashboard

O app LIBRAKIDS acompanha um painel de controle disponível na rede para controle dos dados de usuário e das informações presentes no sistema.

Funcionalidades / Screenshots.

AULA|QUIZ

Acesso as aulas com um vídeo referente a imagem correspondente. Um quiz para memorização é disposto abaixo de cada aula além da opção de fazê-lo correspondente à categoria escolhida.

PONTUAÇÃO

O campo pontuação descreve o número de pontos que o usuário conseguiu durante sua avaliação.

HISTÓRICO

O campo histórico permite ter acesso às escolhas do usuário à saber sobre o que tem errado e acertado durante suas tentativas.
